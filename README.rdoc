== README

API back-end for iLookLikeAnEngineer.com

Stack includes:
* Rails
* RSpec
* Postgres

Dev tasks are logged in Trello project: https://trello.com/b/Bk1TRiiY/illae-tech-things

When contributing:
* Assign Trello card to yourself
* Create a new branch for changes
* Write specs to cover changes made
* When complete create a pull request
